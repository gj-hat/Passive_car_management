每次点击小车按钮都会触发计算

计算逻辑:
1. 遍历:
   1. 如果大于超时时间没有连接 调用移除函数
   2. 处于持续连接的 判断总时长 -> 总时长小于临界值?调用移除函数(移除所有):判断是否大于超时时间
   3. 接上2 大于超时时间 抛出`[小车名,首次连接时间,断开时间]`
   4. 小于超时时间 且处于连接中 忽略交由下次执行

2. 相关函数
   1. 移除函数1    -----  只顺序移除一个
   2. 移除函数2    -----  移除所有
   3. 判断总时长函数   ----  参数(小车名)    返回值 `[小车名,总时长,断连的时间]`
   4. 判断是否大于超时时间函数   ----  参数(`[小车名,断连的时间]`)   返回值   t or f
